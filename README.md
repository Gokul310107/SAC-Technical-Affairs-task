# SAVARA 2026 - Technical Event Hub Landing Page

This is a single-page web application built as an assignment submission for the SAC Technical Affairs Team at IIITDM Kancheepuram.

## Features Included:
- **Dynamic Countdown Timer** configured for the fest timeline.
- **Responsive Event Schedule Grid** featuring tab switching for Day 0, Day 1, and Day 2 (including Ceremonies).
- **Registration Form** with structured inputs and logic that sanitizes/logs entries directly to the developer browser console.
